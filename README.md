# Pet Shop API 

Простое и эффективное FastAPI-приложение для управления каталогом товаров для животных.

##  Функционал
- **CRUD операции** для товаров (создание, чтение, обновление, удаление).
- **Валидация данных**: проверка длины имени (2–80 символов) и цены (>= 0).
- **Гибкая фильтрация**: поиск по диапазону цен и наличию на складе.
- **База данных**: SQLite с использованием SQLAlchemy ORM.

##  Стек технологий
- **Python 3.10+**
- **FastAPI**
- **SQLAlchemy** (ORM)
- **Pydantic** (Валидация)
- **SQLite**
